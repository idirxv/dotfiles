# tempdir.plugin.zsh

`tempdir.plugin.zsh` is an oh-my-zsh extension that allows you to easily create, list, delete, and navigate to temporary directories.

## Installation

1. Clone the repository or download the `tempdir.plugin.zsh` file.
2. Place the `tempdir.plugin.zsh` file in your custom plugins directory, usually `~/.oh-my-zsh/custom/plugins/`.
3. Add `tempdir` to your plugins list in your `.zshrc` file:

```zsh
plugins=(... tempdir)
```
